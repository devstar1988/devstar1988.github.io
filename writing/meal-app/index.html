Meal App

A simple static site that generates random daily meal plans using a small set of pre-defined meals.

  • Website
  • Source Code

Over the weekend (Fri Mar 3 => Sun Mar 5) my brother and I designed/developed a static site that generates a random meal plan for the day. My brother is currently following a strict diet composed of a small set of foods and we thought it'd be cool to create a site that generates a meal plan for him every morning. The site uses a small JSON data file to populate the site's content and has a refresh button that generates a new, random meal plan.
